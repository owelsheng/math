不定积分 ∫f(x)dx

不定积分是求导运算的逆过程，表示找出所有导数为f(x)的原函数集合。其数学定义为：

\[ \text{若 } \frac{d}{dx}F(x) = f(x), \text{则 } \int f(x)dx = F(x) + C \]

其中C为积分常数，代表垂直方向上的无穷多可能性。

核心特点
  • 结果包含任意常数C
  • 表示函数族（曲线族）
  • 与微分方程密切相关
计算方法
  • 基本积分公式
  • 换元积分法
  • 分部积分法
典型应用
  • 求运动物体的位移函数
  • 解微分方程的通解
  • 概率密度函数的累积分布

应用示例

已知速度函数 \( v(t) = 2t \)：

\[ s(t) = \int 2t\,dt = t^2 + C \]

其中C由初始位置确定，体现不同出发点的可能性

定积分 ∫abf(x)dx

定积分表示函数在区间[a,b]上的累积和，由黎曼和极限定义，其核心公式为：

\[ \int_{a}^{b} f(x)dx = \lim_{n\to\infty}\sum_{i=1}^{n}f(x_i^*)\Delta x \]

通过牛顿-莱布尼兹公式可转换为：

\[ F(b) - F(a) \]

其中F(x)是f(x)的原函数

核心特点
  • 结果确定的具体数值
  • 具有线性性和可加性
  • 积分值与变量符号无关
计算方法
  • 牛顿-莱布尼兹公式
  • 数值积分法
  • 对称性简化计算
典型应用
  • 计算曲线围成的面积
  • 求变力做功的总量
  • 概率计算的累积概率

应用示例

计算抛物线 \( y = x^2 \) 在[0,2]区间下的面积：

\[ \int_{0}^{2} x^2 dx = \left.\frac{1}{3}x^3\right|_{0}^{2} = \frac{8}{3} \]

结果精确表示该区域的面积值
